# 智能选品系统 - 分期规划

## v1.0（选产品 + 选品结果池 + 竞品洞察 合并交付）

状态：**立项完成，进入研发**
模式：独立 vibe coding，架构师 + 后端开发
目标：深圳团队内测，验证选品效率提升

### 模块结构

```
选产品（筛选漏斗）→ 选品结果池 → 竞品洞察
```

### 选产品模块

- 卖家精灵实时 API 多维度条件筛选（站点、类目、价格、BSR、销量、评分、评论数、上架时间、增长率、尺寸分段、配送方式）
- 筛选方案：预设方案 + 自定义保存（支持私有/组共享/系统级）
- 结果列表展示（不含顶部统计、不含合规风险/认证需求字段）
- 结果列表工具栏：手动勾选触发侵权词检测 + 以图搜图查重
- AI 评分报告（待确认打分算法）
- 利润估算（待确认公式）

### 选品结果池

- 多维度字段展示 + 列选择器（popover 形式，ERP 交互规范）
- 推送状态管理：待推送 → 推送成功/推送失败/作废
- 推送操作：一键推送 ERP 产品库 + 推送日志查看
- 顶部统计：已推送 / 已作废 / 推送并开品完成
- 不含上级评估流程（专员自主推送）

### 竞品洞察（Tab 切换结构）

**异动机会 Tab**
- 响应策略：路径A（海外仓有相似品→建议通知运营）/ 路径B（无相似品→一键加入结果池，90天窗口）
- 筛选：异动类型、类目、站点
- 数据表：竞品、类目、异动类型、原日均销量、当前状态、市场空缺、响应方案、操作
- 监控设置入口

**BSR 榜单监控 Tab**
- 按监控人切换（下拉选择不同人创建的监控任务）
- 公共统计：稳定在榜、新品进入、新品进入率、卖家国籍占比、搜索供需比、有品牌占比
- ASIN 明细表：排名变化、异常类型、操作（评论洞察 / 趋势洞察）

**刊登链接监控 Tab**
- 按监控人切换
- 公共统计：监控 ASIN 数、排名异动、差评告警、状态正常
- ASIN 明细表：BSR/价格/评分/评论/状态、操作（评论洞察 / 趋势洞察）

**监控管理 Tab**
- 每人最多 5 个监控任务
- 表格管理：监控名称、类型、范围、频率、归属、本月异动、状态、操作（编辑/暂停/删除）
- 新建监控入口

### 不含（明确排除）

- 运营工单对接 ERP（预留占位，v1.0 仅"建议通知运营"）
- 1688 API 自动寻源
- 多类目并行
- 自动开品
- 合规风险/认证需求字段（无数据源）
- AI 评分报告打分算法（待梳理）

### 数据源

- 卖家精灵：实时接口（文档已整理，见 docs/data-sources/sellersprite-api-full.md）
- ProBoost：BSR 快照 + 评论（洽谈中）

### 研发团队

- 架构师 × 1
- 后端开发 × 1（Java / Spring Boot 3.x）
- 开发模式：独立 vibe coding

## v1.1（竞品洞察深化 + ERP 闭环）

前置条件：v1.0 内测验证 + ProBoost 数据采购落地

- 运营工单对接 ERP（异动机会→自动创建运营工单）
- 评论情绪洞察 → 图文优化建议 → 推送 ERP ECN 工单
- 利润计算精细化（FBA 费用 API 对接）
- AI 评分报告算法落地
- 合规风险/认证需求字段（待第三方知产审核数据源确认）

---

## v2.0（供应链 + 多类目扩展）

前置条件：v1.1 效果验证 + ERP ECN 工单接口开放

- 1688 供应链对接（以图搜图 API 回流 + 供应商匹配）
- 多类目扩展（规则模板 + 类目差异化权重）
- 选品结果与 ERP 开品流程深度打通
- BSR 快照对比 → 竞品异常自动监测 + 告警推送
- 卖家监控（竞争对手店铺动态）

---

## v3.0（智能化 + 自动化）

- 补货拦截联动（目的国电气标准校验）
- ML 模型替代规则引擎（需积累标注数据）
- 自动开品流程（高置信度选品结果直通 ERP）
