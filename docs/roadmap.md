# 智能选品系统 - 分期规划

> v1.0 模块详细设计见 Demo 原型 + `requirements/` 目录。本文件仅做版本摘要。

---

## v1.0 — 选品闭环 + 竞品洞察（2026.07 — 2026.09）

**状态**：立项完成，进入研发
**团队**：架构师 × 1 + 后端 × 1（Java / Spring Boot 3.x，vibe coding）
**目标**：深圳团队内测，验证选品效率提升 30%+

### 模块

| 模块 | 核心能力 |
|------|----------|
| 选产品 | 卖家精灵 API 多维度筛选 + 筛选方案保存 + 手动风控（侵权词/以图搜图） |
| 选品结果池 | 多维度展示 + 推送状态管理 + 一键推送 ERP |
| 竞品洞察 | 异动机会 / BSR 榜单监控 / 刊登链接监控 / 监控管理（4 Tab） |

### 不含

运营工单对接、1688 寻源、AI 评分算法、自动开品、合规风险字段

### 数据源

- 卖家精灵实时 API（文档：`data-sources/sellersprite-api-full.md`）
- ProBoost BSR 快照 + 评论（洽谈中）

## v1.1 — 竞品深化 + ERP 闭环（2026.10 — 2026.11）

前置：v1.0 内测通过 + ProBoost 采购落地

## v2.0 — 供应链 + 多类目（2027 Q1）

## v3.0 — 智能化 + 自动化（持续迭代）
